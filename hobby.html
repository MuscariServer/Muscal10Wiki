アスレチック

  • 場所：アスレチックワールド
  • Mod要素を含めたアスレチックが楽しめます。
  • 初回クリアでガチャが回せるコインが獲得できます。
  • まだまだ数が少ないので製作者募集中です。

ブラックジャック

  • 場所：カジノワールド
  • 最低人数：２人
  • マイクラ内でブラックジャックが楽しめます。
  • お金を賭けてプレイヤー同士で遊ぶことができます。
  • ※テーブルのオーナーがディーラーになります。ディーラーはヒットやスタンドといった行動はできません。
  • コマンド一覧
コマンド 詳細
//bj ブラックジャックメニューを開きます。
/table create <名前> <バイ・イン> bj テーブルを作成します。ゲームはテーブルで行われます。
バイ・インの金額がスタック(テーブル内で使えるお金)になります。
/table open テーブルを公開し、他プレイヤーが参加できるようにします。
/table list <テーブルID> テーブルのリストを表示します。
/table join <テーブルID> 指定したテーブルIDのテーブルに参加します。
テーブルIDはテーブルのオーナーがテーブルを公開するとチャット欄に表示されます。
/table rebuy <金額> 自分のスタックにカジノマネーから金額を追加します。
/table withdraw <金額> 自分のスタックからカジノマネーに指定した金額引き出します。
/table set <設定> <数値/true or false> テーブルの設定ができます。設定リストは/table listsettingsで参照してください。
/table bet <金額> 指定した金額ベットします。
/table start ゲームを開始します。少なくとも参加者のうち１人がベットしている必要があります。
/table setでオートスタートの設定もできます。
/table delete オーナーが自分のテーブルを削除します。削除するにはそのテーブルに参加している必要があります。

スロットマシン

  • 場所：カジノワールド
  • １人でも楽しめるこのスロットマシンは、２種類のスロットマシンがあります。
  • ・ノーマル
  • 最高100倍の賞金を得ることができる上、スペードにに比べて安定して高い倍率の絵柄が揃います。
  • ・スペード
  • 最高500倍と莫大な賞金を得ることができますが、倍率が低い絵柄が揃いがちです。

ガチャ

  • 場所：ロビー
  • 投票の報酬で貰えるコイン10個で1回ガチャを回せます。
  • Tech LootbagとMagic Lootbagの2種類あり、取得したスキルツリーに合わせてガチャを回せます。
