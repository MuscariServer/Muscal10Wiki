・Simple Storage Network

　　
名前 画像 作り方 説明
Storage Cable 石のハーフブロック*６
鉄インゴット*２		 ネットワークを繋ぐためのケーブル。
Storage Link Cable Storage Cable*４
チェスト*１		 ネットワークにストレージを検出させるケーブル。
１ケーブルにつき１つのストレージにしか接続できない。
Storage Export Cable Storage Cable*４
ピストン*１		 ネットワークからアイテムを搬出するケーブル。
GUI中段に搬出したいアイテムを置く。
Storage Import Cable Storage Cable*４
ホッパー*１		 ネットワークからアイテムを搬入するケーブル。
GUI中段に搬出したいアイテムを置く。
ホワイトリスト/ブラックリスト変更可。
Processing Cable 金塊*４
Storage Cable*４
オブザーバー*１		 自動処理を行うケーブル。
左側に材料、右側に完成品を設定し、接続した機械のどの面から搬入/搬出するかを設定する。
使用にはControllerが必要。
Speed Upgrade レッドストーンブロック*２
金インゴット*２
Storage Cable*１		 Storage Export/Import Cableに使用できる。
アイテムの搬入出速度を上昇させる。
Operation Upgrade レッドストーンコンパレーター*２
レッドストーン*２
Storage Cable*１		 Storage Export/Import Cableに使用できる。
レッドストーン制御が可能になる。
Operation Upgrade レッドストーンブロック*２
ブレイズパウダー*２
Storage Cable*１		 Storage Export/Import Cableに使用できる。
アイテムをスタック単位で搬出させる。
Operation Upgrade レッドストーンコンパレーター*２
Storage Cable*１		 Storage Export Cableに使用できる。
搬出先に１つ以上搬出対象のアイテムがあった場合、アイテムを搬出しない。
Storage Network Master クォーツブロック*４
Storage Cable*４
ダイヤモンド*１		 接続された各ストレージをネットワークで接続し、相互利用可能な状態にする。
１つのネットワークに１つまで置ける。
Storage Request Table 金インゴット*４
Storage Cable*４
作業台*１		 接続されたネットワークの全てのストレージにアクセスできるブロック。
Controller ラピスラズリブロック*４
Processing Cable*４
黒曜石*１		 自動処理を管理する。
２つのモードがあり、左側のチェックマークを押すことで切り替えられる。
【Always Active】常時指定された処理を実行する。
【Only Process the Request Number】右の数値の回数分残りの処理を行う。
Storage Remote Storage Import Cable*２
金インゴット*４
Storage Cable*１
Storage Export Cable*２		 遠隔でネットワーク内のストレージにアクセスできるようになるアイテム。Storage Network Masterに右クリックでリンクさせることができる。
範囲は半径64ブロックまで。
Simple Storage Remote スライムボール*４
グロウストーン*１
金インゴット*２
Storage Network Master*１		 遠隔でネットワーク内のストレージにアクセスできるようになるアイテム。Storage Network Masterに右クリックでリンクさせることができる。
同ディメンション内ならどこからでもアクセスできる。
クラフティンググリッドは使えない。
Upgraded Storage Remote エンダーパール*４
グロウストーン*１
金インゴット*２
Storage Network Master*１
マグマブロック*１		 遠隔でネットワーク内のストレージにアクセスできるようになるアイテム。Storage Network Masterに右クリックでリンクさせることができる。
同ディメンション内ならどこからでもアクセスできる。
Upgraded Storage Remote エンダーパール*４
ブレイズロッド*１
プルパァブロック*２
ネザースター*１
マグマブロック*１		 遠隔でネットワーク内のストレージにアクセスできるようになるアイテム。Storage Network Masterに右クリックでリンクさせることができる。
あらゆる場所からネットワークにアクセスできる。
